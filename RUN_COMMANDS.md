# Run Commands

From repository root:

```powershell
cd "D:\TRAE项目\stocks_lxy-main"
node scripts/dev.js
```

Or:

```powershell
cd "D:\TRAE项目\stocks_lxy-main"
npm.cmd start
```

Project source directory:

```powershell
cd "D:\TRAE项目\stocks_lxy-main\stock-board-journal"
node scripts/dev.js
```

Default URLs:

- Frontend: `http://localhost:3002`
- Backend health: `http://localhost:3001/api/health`
